## 📌 About This Project

This is not just another list of SQL tutorials.

I spent weeks researching, testing, and curating the **best SQL learning resources** available — both free and paid — and organized them into a structured, **16-week learning roadmap** with a fully interactive website.

Every resource includes:
- ✅ Why it's worth your time
- ✅ Time commitment
- ✅ Free or paid label
- ✅ Star rating
- ✅ Direct link

## 📚 What's Inside

The roadmap is divided into **5 phases** covering the complete SQL learning journey:

---

### ◈ Phase 1 — Beginner `Weeks 1–3` `~45 hrs`

> **Goal:** Understand SQL syntax, tables, filtering, joins, and aggregates

| Resource | Type | Cost |
|---|---|---|
| freeCodeCamp SQL Full Course | Video | 🆓 Free |
| Khan Academy SQL | Interactive | 🆓 Free |
| The Complete SQL Bootcamp (Udemy) | Video | 💳 ~$15 |
| SQLZoo | Practice | 🆓 Free |
| Mode SQL Tutorial | Practice | 🆓 Free |
| DB Fiddle — Browser Sandbox | Tool | 🆓 Free |
| Learning SQL — O'Reilly Book | Book | 💳 ~$50 |
| PostgreSQL + pgAdmin | Tool | 🆓 Free |

**Topics:** `SELECT` `WHERE` `AND/OR` `ORDER BY` `GROUP BY` `HAVING` `JOINS` `Aggregate Functions`

---

### ◆ Phase 2 — Intermediate `Weeks 4–7` `~60 hrs`

> **Goal:** Master CTEs, window functions, database design, and indexing

| Resource | Type | Cost |
|---|---|---|
| Window Functions — Socratica (YouTube) | Video | 🆓 Free |
| Advanced SQL for Data Scientists (LinkedIn) | Video | 💳 Premium |
| PostgreSQL Advanced SQL Queries (Udemy) | Video | 💳 ~$15 |
| LeetCode Database Problems | Practice | 🆓 Free |
| StrataScratch — FAANG Questions | Practice | 🆓 Free |
| Use The Index, Luke! | Reference | 🆓 Free |
| SQL Antipatterns — Book | Book | 💳 ~$45 |
| Database Design for Mere Mortals — Book | Book | 💳 ~$55 |

**Topics:** `Subqueries` `CTEs` `Window Functions` `ROW_NUMBER` `RANK` `LAG/LEAD` `Normalization` `1NF/2NF/3NF` `B-Tree Indexes`

---

### ◉ Phase 3 — Advanced `Weeks 8–11` `~50 hrs`

> **Goal:** Transactions, stored procedures, triggers, and performance tuning

| Resource | Type | Cost |
|---|---|---|
| Database Transactions ACID — Hussain Nasser | Video | 🆓 Free |
| PostgreSQL PL/pgSQL Complete Course (Udemy) | Video | 💳 ~$15 |
| SQL Server Stored Procedures (Pluralsight) | Video | 💳 ~$29/mo |
| DataLemur — FAANG SQL Questions | Practice | 🆓 Free |
| HackerRank SQL Hard | Practice | 🆓 Free |
| SQL Performance Explained — Book | Book | 💳 ~$30 |
| PostgreSQL Up & Running — Book | Book | 💳 O'Reilly |
| DBeaver Community Edition | Tool | 🆓 Free |

**Topics:** `ACID` `BEGIN/COMMIT/ROLLBACK` `Isolation Levels` `Deadlocks` `Stored Procedures` `Functions` `Triggers` `BEFORE/AFTER` `EXPLAIN ANALYZE`

---

### ◎ Phase 4 — Expert `Weeks 12–16` `~70 hrs`

> **Goal:** Architecture, data warehousing, performance at scale, real-world projects

| Resource | Type | Cost |
|---|---|---|
| CMU 15-445 Database Engineering (YouTube) | Video | 🆓 Free |
| Data Warehouse Fundamentals (Udemy) | Video | 💳 ~$15 |
| PGExercises — PostgreSQL Deep Dives | Practice | 🆓 Free |
| LeetCode Hard SQL | Practice | 🆓 Free |
| Designing Data-Intensive Applications (DDIA) | Book | 💳 ~$60 |
| The Data Warehouse Toolkit — Kimball | Book | 💳 ~$65 |
| HackerRank SQL Certificate | Certification | 🆓 Free |
| Google Data Analytics Certificate | Certification | 💳 ~$49/mo |
| Microsoft DP-900 Azure Data Fundamentals | Certification | 💳 $165 exam |

**Topics:** `Query Optimization` `Execution Plans` `Partitioning` `Materialized Views` `Star Schema` `Fact & Dimension Tables` `SCD Type 2` `Data Warehouse Design`

---

### ∞ Phase 5 — Ongoing `Forever`

> **Goal:** Stay current, build real things, contribute and network

| Resource | Type |
|---|---|
| DBA Stack Exchange | Community |
| r/SQL — Reddit | Community |
| Planet PostgreSQL | Blog Feed |
| Haki Benita's Blog | Blog |
| Kaggle Datasets + SQL | Projects |
| GitHub Open Source | Contributing |
